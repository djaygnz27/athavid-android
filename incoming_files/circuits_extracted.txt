The Schedule Performance Index (SPI) is calculated using the
formula ; SPI= Earned Value (EV)/Planned Value (PV).
It measures project schedule efficiency by comparing the value of work completed (earned Value) to the
value of work planned to be completed at a specific point in time (planned Value). A score of 1.0 means
the project is on schedule, while a score less than 1.0 indicates the project is behind schedule, and a score
SPI = EV ÷ PV
greater than 1.0 means it is ahead of schedule.

• Earned Value (EV):% actually completed by that date.

SPI = EV ÷ PV
