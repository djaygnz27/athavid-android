**What was said:**
- JMUX today supports serial connectivity directly into the MUX
- Nokia ONLY provides IP connectivity — no native serial support
- This requires the T&D / Relay Protection team to build an intermediary solution to convert serial → IP
- This solution (Iniven RC-30) is being implemented at Phase 2 critical sites
- Cost was budgeted at ~$20K originally — actual cost is ~$500K — a major variance

**Irving's observation:**
> *"Serial to IP conversion, to me, was a basic fundamental requirement that should have been caught from the beginning."*

**Jay / team response:**

- The level of effort and financials only became clear during Phase 2 detailed design
- This is a common pattern: detailed site surveys reveal the true scope and cost

**Key message for future reporting:**
- Phase 3 team is already proactively engaging stakeholders early to avoid similar surprises
